1. What is This !?　～ これってなんやねん

　これは Windowsのシャットダウン操作をもっとわかりやすくして～な～ と思った結果テンションで作りました。
　WindowsのSHUTDOWNコマンドをGUi操作でやりたかっただけなので特別な機能はないです。

---------------------------------------------------------------------------------------------------
2. 必要スペック

　必要最低スペック：
    OS: Windows 7以降（SHUTDOWNコマンドが使えるWindowsだったらおｋ）
    　Windows 7にて動作を確認しています。Windows 7で 再起動して回復環境を開く ができないことが確認されています。
    　これにより、Windows 7以前では 再起動して回復環境を開く ができないと考えられます。
    　9x系とかすっごい古いNT(NT 3.x とか NT 4.xとか)、XP、VistaではそもそもEasyShutdownが動くかわかりません。（多分動きます）
    ほか: OSの最低スペックで使える（と思う）
    同封のVC_redistなんとか.exeを実行したらVC++の再頒布可能パッケージの中身がインストールされます。
    EasyShutdownが動かないときに活用ください。

---------------------------------------------------------------------------------------------------
3. 使い方（意味浅）　～ べっ、別に、特別な意味なんてないんだからねっ！

　実行するとウィンドウが出ます。
　ボタン、チェックボックス、メニューバーがあるんでそれを見てなんとなく あーはいはい、あれね。 と察してください。
